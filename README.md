# Minimax
Algoritmo Minimax en Python de estructura modular para mejor entendimiento, orden y visualizacion de las partes, ya sea como la dimension del tablero
la dimension de las caracteristicas basicas de nuestros objetos en la clase padre animal y heredando caracteristicas a las clases hijas: Cat (gato), 
Mouse(ratón), entre las cosas que más costaron son la implementación del algoritmo Minimax, ya que no funcionaba como queria, por ejemplo, se desplazaba
de forma errónea o el gato jamás atrapaba al ratón
